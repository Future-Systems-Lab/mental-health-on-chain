## What This Is

Mental health concepts delivered as smart contracts — each one a standalone psychological framework enforced by code. No apps to download, no accounts to create. Just a wallet, a testnet, and a ritual.

Built for developers, Web3 natives, and the digitally curious. Participate directly from GitHub.

Each contract in this series:
- Is deployed and verified on Sepolia testnet
- Costs nothing to use (free test ETH)
- Teaches blockchain by doing something meaningful
- Cites a real psychological framework
- Includes a full beginner tutorial

## How to Participate

Every contract uses the same entry point:

1. **Install MetaMask** — [metamask.io](https://metamask.io)
2. **Switch to Sepolia testnet**
3. **Get free test ETH** — [sepoliafaucet.com](https://sepoliafaucet.com)
4. **Open the contract folder above** and follow the README

No coding required. You only need a browser and a wallet.
