# Privacy Policy - mental-health-on-chain Campaign

## PHI Protection (sovereign health governance Compliance)

This campaign **does not**:
- Collect personal health information (PHI)
- Store medical data
- Track individual health outcomes
- Create health records
- Diagnose or treat conditions

## Educational Content Only

All messaging:
- References psychological frameworks (Jungian, somatic, attachment theory)
- Discusses blockchain/Web3 technology
- Educates on orthomolecular principles conceptually
- **Does NOT** provide medical advice
- **Does NOT** constitute session

## User Consent

- All interactions are voluntary
- Users control their own data on-chain
- No personal information stored in campaign logs
- GitHub metrics track repo activity only, not user behavior

## Data Handling

- `.env` file: API keys only (encrypted at rest)
- Logs: Campaign execution records, not user data
- Telegram reports: Campaign status, not user information
- GitHub: Public repo metrics only

## Disclaimer

This project is educational and informational. It does not constitute medical, psychological, or financial advice. Users seeking healthcare should consult licensed professionals.
